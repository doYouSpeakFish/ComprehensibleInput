# Comprehensible Input

Children learn language by listening to comprehensible input.  They don't learn by memorizing
vocabulary lists or grammar rules.  They don't learn by doing grammar drills or by taking
grammar tests.  They learn by listening to comprehensible input.

The goal of this app is to provide sources of comprehensible input for language learning.

[Now available on Google Play](https://play.google.com/store/apps/details?id=in.comprehensible)

## Ktor backend module

The repository includes a JVM backend module at `:backend` with a single health check endpoint.

### Run the backend locally

```bash
./gradlew :backend:run
```

The service starts on `http://localhost:8080`.

### Health check

```bash
curl http://localhost:8080/health
```

Expected response:

```text
ok
```


### Coverage snapshot workflow

Use these tasks for debug coverage markdown snapshots (generated by the custom Kover markdown plugin):

```bash
# Verify snapshots match current coverage output
./gradlew :app:koverMarkdownSnapshotCheckDebug

# Regenerate stored debug snapshots when coverage output changes
./gradlew :app:koverMarkdownSnapshotDebug

# Re-verify after regeneration
./gradlew :app:koverMarkdownSnapshotCheckDebug
```
